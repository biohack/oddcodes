* ABOUT *

Initial proteomic analysis of unknown amino acid sequences

Source code can be obtained here:
http://cloud-131-215.diagcomputing.org/shwang26/final_submission.tar

Demo with pre-uploaded files:
http://cloud-131-215.diagcomputing.org/shwang26/final.html

Utilizes tools published at:

http://doc.bioperl.org/releases/bioperl-1.4/Bio/Tools/Run/RemoteBlast.html
* Bio::Tools::Run::RemoteBlast: object for remote execution of NCBI BLAST

http://doc.bioperl.org/releases/bioperl-1.4/Bio/Tools/OddCodes.html
* Bio::Tools::OddCodes: object holding alternative alphabet coding for one protein sequence

* REQUIREMENTS *

Storage is minimal.

Recommended memory/cpu is 1gb, 1.8ghz per simultaneous user.

* DETAILED USAGE *

1. Enter in any 20-mer amino acid sequence in `blastp2mysql` form. Press `submit` button. Close pop-up alert.

2. Wait until table consisting of RemoteBlast hits show. Check the checkbox next to results that you.re interested in seeing odd code\
 for. To save results, press `save selected results` button.

3. After about 30 seconds, enter any term shown in `Description` of RemoteBlast results and press `search` button.
